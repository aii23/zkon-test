# Mina zkApp: Zkon_test

This template uses TypeScript.

## How to install

```sh
npm install
```
## How to build

```sh
npm run build
```

## How to deploy zkrequest

```sh
DEPLOYER_KEY=EK... npm run deployZkRequest```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
